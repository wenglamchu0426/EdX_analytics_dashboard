**EdX Analytics Dashboard on Tableau Public**

I created a dashboard for EdX courses showing global user demographics, course enrollment patterns, and engagement trends using data visualizations on Tableau Public. 

Data: edx.org Academic Year 2012-2013 data from Harvard Dataverse (https://dataverse.harvard.edu/dataverse/mxhx).

Usage: Download the .twbx file and open in Tableau Public or Tableau desktop to view and interact with dashboard.
